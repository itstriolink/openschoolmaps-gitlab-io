NOTE: Avec cette feuille de travail, tu vas apprendre comment changer le style de la carte sur openstreetmap.org, comment utiliser la planification d'itinéraire et comment partager les positions trouvées sur la carte avec les autres.

== Style de carte

Malgré le nom et la vue proéminente sur openstreetmap.org, OpenStreetMap n'est pas seulement une carte. 
Il s'agit d'une collecte de données géographiques détaillées à partir desquelles les ordinateurs peuvent 
créer des cartes. Les données peuvent non seulement créer des cartes mais aussi - sur la base des mêmes 
données - des cartes pour différents usages.

Le site Web de l'OSM.org offre déjà un choix de cartes. Ce que tu vois en premier est le style de carte 
standard OpenStreetMap. Si tu cliques sur l'icône « Couches » avec les surfaces empilées 
image:osm_editieren/osm_bearbeiten/osm_ebene_icon.PNG["Flächen-Icon", 20, 20] au-dessus du 
image:osm-org_als_kartenviewer/info_button.PNG["Info Button", 20 , 20]-bouton à droite, tu 
peux changer entre ces différentes représentations cartographiques.

== Navigation (planification d'itinéraire)

Avec la collection de géodonnées OpenStreetMap, les ordinateurs peuvent faire plus que « simplement » 
dessiner des cartes ! La fonction de recherche, que nous avons déjà examinée, utilise les mêmes données 
et ajoute des données supplémentaires libres, par exemple provenant de WikiData (un projet frère de Wikipedia), 
pour compléter la recherche.

Trouver des chemins d’A à B est également possible avec les données OpenStreetMap. Ça marche comme cela:

Clique sur le image:osm-org_als_kartenviewer/navigation_button.PNG["Navigation Button", 25, 25]-bouton à côté du 
champ de recherche. Tu peux maintenant effectuer une recherche par texte (par ex. adresse) pour le départ et la 
destination, ou faire glisser l’épingle rouge et verte du formulaire de recherche à l'endroit correspondant sur la carte.

A l'aide du menu déroulant situé sous les informations de départ et de destination, tu peux sélectionner 
le moyen de transport / type de déplacement (voiture, vélo ou à pied) et le programme de recherche 
d'itinéraire (en jargon technique : l'algorithme de routage).

.Stations intermédiaires
[NOTE]
====
Malheureusement, il n'est pas possible de spécifier des arrêts intermédiaires ou des "via" sur openstreetmap.org. 
Sur http://routing.osm.ch, il existe une autre option de planification d'itinéraire basée sur OpenStreetMap, 
mais ça fonctionne que pour les itinéraires en Suisse.
====
